# AMS Mobile Site Register App

Standalone read-only phone prototype for the AMS Site Register.

The main AMS master HTML remains the source of truth. This app is a separate trial version for viewing current register information on a phone or tablet.

## Current Prototype

- Read-only mobile app shell.
- Search assets.
- Filter by asset type.
- View simplified mobile map markers.
- Open asset detail cards.
- Show a compact data status card.
- Keep mobile app changes separate from the master database.

## GitHub Pages Trial

Upload these files to a GitHub repository:

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`
- `README.md`

Then enable GitHub Pages from the repository settings and choose the main branch/root folder.
